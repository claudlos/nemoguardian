Nemoguardian is an open-source, self-hosted moderation template for community and organization operators. It provides code, documentation, bot adapters, audit tooling, and optional GPU-credit workflows for running moderation infrastructure with your own accounts and policies.

Use Of The Project

You are responsible for how you deploy Nemoguardian, which communities you connect it to, which providers you use, and which moderation actions you enable. Always test in shadow or review mode before enabling automatic deletes, timeouts, or other enforcement actions.

Accounts And Provider Credentials

Nemoguardian is designed for self-hosting. You bring your own Discord, Stripe, GPU provider, model API, and other third-party credentials. Keep private keys and tokens outside public repositories.

GPU Credits

GPU credits are digital service credits used by the project workflow to reserve rented GPU capacity for self-hosted moderation runs. Stripe is used to process payments. GPU credits do not make Stripe the GPU provider, and they do not transfer ownership of GPU hardware.

Acceptable Use

Do not use Nemoguardian to violate law, evade platform rules, harvest private data, target people unfairly, or run abusive automation. You are responsible for reviewing model outputs and configuring human oversight for high-impact moderation decisions.

No Warranty

The open-source code is provided under the Apache-2.0 license. The software and documentation are provided as-is, without guarantees that a model decision, provider integration, or moderation action will be accurate, uninterrupted, or suitable for every community.

Support

Support, setup questions, and bug reports are handled through GitHub Issues.
